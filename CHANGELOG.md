# 변경 이력 (Changelog)

이 프로젝트의 모든 주목할 만한 변경 사항은 이 파일에 기록됩니다.

이 프로젝트는 [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) 형식을 따르며,
[유의적 버전(Semantic Versioning)](https://semver.org/spec/v2.0.0.html)을 준수합니다.

## [1.0.0] - 2026-04-06

### 추가됨 (Added)
- **우클릭 컨텍스트 메뉴**: Windows 탐색기에서 `.heic` 및 `.heif` 파일에 대해 "HEIC to JPG" 변환 옵션 추가.
- **병렬 처리 지원**: 고성능 변환을 위해 최대 4개 파일을 동시에 처리하는 멀티스레딩 구현.
- **다국어 지원 (i18n)**: 시스템 언어 설정에 따라 한국어와 영어 UI가 자동으로 전환되도록 구현.
- **진행 추적 UI**: 진행 표시줄과 각 파일별 상태(성공, 실패, 건너뜀)를 확인할 수 있는 상세 리스트 뷰 제공.
- **스마트 덮어쓰기 대화상자**: 이미 파일이 존재할 경우 "전체 적용" 기능을 포함한 사용자 친화적인 알림 창 구현.
- **안정적인 종속성 로딩**: 셸 확장 환경에서 외부 라이브러리가 원활하게 로드되도록 특수 어셈블리 리졸버(Assembly Resolver) 로직 추가.
- **강력한 이름(Strong Name) 서명**: 빌드 스크립트(`build.ps1`)를 통해 모든 종속성과 어셈블리에 대한 자동 서명 처리.
- **자동 등록 스크립트**: 셸 확장을 쉽게 설치하고 제거할 수 있는 PowerShell 스크립트 제공.
